# 💸 Money Transfer API

## Overview
REST API for managing users, accounts, and money transfers.

## Features
- Create users and accounts
- Transfer money between accounts
- Transaction history
- Validation & business rules
- Global error handling

## Tech stack
- Java 21
- Spring Boot
- PostgreSQL
- Spring Data JPA

## How to run
mvn spring-boot:run

## API docs
http://localhost:8080/swagger-ui.html
